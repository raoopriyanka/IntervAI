# IntervAI – AI Mock Interview Platform 🚀

## Overview
IntervAI is a full-stack AI-powered web application designed to help students practice mock interviews and improve their skills through intelligent feedback.

## Tech Stack
- Frontend: React (Vite), Tailwind CSS
- Backend: FastAPI (in progress)
- AI/ML: Speech & NLP (planned)

## Current Progress (Phase 1 – Frontend)
- Landing Page
- Authentication UI
- Dashboard UI
- Interview Setup Screen
- React Router integration

## Upcoming Features
- Backend API
- AI-based feedback
- DSA coding environment
